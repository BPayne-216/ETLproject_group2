Movie Analysis by Ratings

by Ben Stork, Crisharon Beale, Robert Payne, Julie Pyle

Our team created a movie information data frame in SQL (postgres) that allows users to quickly and conveniently search for the top movies by ratings. The following rating systems were utilized:
1. Rotten Tomatoes
2. IMDB
3. User ratings

We analyzed the Top 5 movies for each rating system by:
1. Genre
2. Release Year
3. Age

Lastly, we analyzed what streaming platform a movie can be found.
