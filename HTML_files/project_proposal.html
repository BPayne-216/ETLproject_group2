Project Proposal

by Ben Stork, Crisharon Beale, Robert Payne, Julie Pyle

Create a movie information data frame in SQL that allows users to quickly and conveniently search for the top movies based on IMDb and Rotten Tomatoes ratings.

We will use the following movie data sets from Kaggle: movies on streaming networks dataset and the movie metadata set.

Users will be able to utilize the dataframe in SQL to complete any of the following analyses (and more):
- Top movie ratings from Rotten Tomatoes and IMDb by genre
- Top movie ratings from Rotten Tomatoes and IMDb by age category
- Top movie ratings from Rotten Tomatoes and IMDb by year
- What streaming service to find the movie
