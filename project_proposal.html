Project Proposal

by Ben Stork, Crisharon Beale, Robert Payne, Julie Pyle

Create a movie information data frame in SQL that allows users to quickly and conveniently search for the following:

1. Compare movie ratings by release year and find the top 5 rated movies based on the combined data sets (ratings on Rotten Tomatoes, IMDb and user reviews).
2. Compare movie ratings by genre looking at the top 5 rated movies based on the combined data sets (ratings on Rotten Tomatoes, IMDb and user reviews.
3. Compare top 5 movies by target age group of movies.
4. Inform the consumer where to stream a movie.
